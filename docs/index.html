Modelo mental clave

IA como herramienta vs IA como agente

Esta diferencia es importante para clase: no es lo mismo usar un chat para ayudarte que montar un sistema que planifica pasos, llama herramientas y ejecuta acciones.

IA como herramienta

Te ayuda a escribir, resumir, explicar o revisar, pero una persona decide y ejecuta.

  • Menor riesgo operativo
  • Ideal para empezar
  • Mas facil de revisar

IA como agente

Recibe un objetivo, usa herramientas, encadena pasos y necesita mas control y observacion.

  • Mayor autonomia aparente
  • Mas superficie de fallo
  • Exige politicas y verificacion
